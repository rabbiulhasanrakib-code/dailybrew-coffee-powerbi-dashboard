# Dataset Description

This dashboard uses a retail coffee shop sales dataset covering transactions between January 2017 and April 2019.

## Dataset Overview

The dataset contains transactional records for multiple coffee shop stores including information about products, customers, staff, and sales.

## Key Entities

The dataset includes information related to:

- Sales transactions
- Product categories and pricing
- Store locations
- Customer purchases
- Staff performance

## Key Fields Used in the Dashboard

Some important fields used in the analysis include:

- Revenue
- Quantity sold
- Product group
- Store ID
- City
- Customer gender
- Staff ID
- Order date

## Analytical Purpose

The dataset enables analysis of:

- Revenue trends over time
- Store-level performance
- Product category profitability
- Customer purchasing behavior
- Staff sales performance

The dashboard transforms these transactional records into interactive visual analytics for business insights.
